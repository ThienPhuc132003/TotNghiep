🧪 Hướng Dẫn Kiểm Tra

Đăng nhập với quyền Admin

Truy cập trang đăng nhập và đăng nhập với tài khoản Admin để có quyền truy cập các trang thống kê.

Truy cập từng trang thống kê

Click vào từng link ở trên để kiểm tra giao diện và chức năng xuất Excel.

Kiểm tra nút Export

Tìm nút với icon Excel (màu xanh lá). Hover để xem tooltip "Xuất Excel (Ctrl+E)".

Test xuất Excel

Click nút xuất hoặc nhấn Ctrl+E để tải file Excel. Kiểm tra file có chứa:

  • Logo Van Lang University ở góc trái trên
  • Header với tên trường và khoa
  • Tiêu đề báo cáo và thời gian
  • Bảng thống kê tóm tắt với màu nền
  • Dữ liệu được định dạng đẹp với viền
  • Footer với ngày xuất và địa điểm
Test với dữ liệu khác nhau

Thử thay đổi bộ lọc thời gian và xuất lại để kiểm tra tính nhất quán.

🔧 Trouble Shooting

Nếu gặp lỗi: 1. Kiểm tra console browser để xem lỗi chi tiết 2. Đảm bảo ExcelJS đã được cài đặt: npm list exceljs 3. Kiểm tra file logo có tồn tại: src/assets/images/logo_vanlang.png 4. Restart development server nếu cần 5. Clear browser cache và thử lại
