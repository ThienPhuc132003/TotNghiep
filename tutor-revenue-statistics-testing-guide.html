🔧 API Integration Details

API Request Structure

GET /api/manage-payment/search-with-time-by-tutor Query Parameters: - page: Current page number (1-based) - rpp: Records per page (5, 10, 20, 50, 100) - timeRange: Filter period (day, week, month, year) - sort: JSON string [{"key": "createdAt", "type": "DESC"}]

Expected Response Structure

{ "success": true, "data": { "items": [ { "id": "transaction_id", "coinOfUserPayment": 100000, "coinOfTutorReceive": 90000, "coinOfWebReceive": 10000, "paymentType": "ONLINE", "createdAt": "2024-01-15T10:30:00Z", "tutor": { "userId": "tutor_id", "fullName": "Nguyễn Văn A", "email": "tutora@example.com" }, "user": { "userId": "user_id", "fullName": "Trần Thị B", "email": "userb@example.com" } } ], "total": 150, "totalRevenue": 15250000 } }

🐛 Debugging & Troubleshooting

Common Issues & Solutions

  • Charts not loading: Check Chart.js component import and canvas rendering
  • API errors: Verify endpoint URL and authentication tokens
  • Empty data: Check database for tutor payment records
  • Pagination issues: Verify total items count and page calculation
